Embla based carousel

[View documentation](https://mantine.dev/)

## Installation

```bash
# With yarn
yarn add @reactgets/core @reactgets/hooks @reactgets/carousel

# With npm
npm install @reactgets/core @reactgets/hooks @reactgets/carousel
```
