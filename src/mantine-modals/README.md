Manage modals state

[View documentation](https://mantine.dev/)

## Installation

```bash
# With yarn
yarn add @reactgets/hooks @reactgets/core @reactgets/modals

# With npm
npm install @reactgets/hooks @reactgets/core @reactgets/modals
```
